8인 개인전 트레이닝 맵으로 순위에 따라 결과 보상을 받으며 삼도천에 빠질 경우 이전 세이브포인트로 이동하게 되며 유령 상태가 되어 분노/힘 치임/밟기 효과를 받지 않게 됩니다. 삼도천에 빠지지 않고 무사히 골인 지점까지 도착해 주세요!

유의사항

  • 일일 미션은 매일 00시에 초기화되어 다음 날 다시 진행하실 수 있습니다.
  • 달성 보상 중 게임 아이템은 게임 내 마이룸으로 지급됩니다.
  • 톱(=웹 포인트)은 1일 최대 18개까지 획득 가능하며, 이벤트 종료 시 자동 소멸됩니다.
