닉네임 응모 횟수 추가 보상

  • 응모 횟수 10회 - 스윗바닐라 코스튬 박스 1개
  • 응모 횟수 20회 - 17주년 코인 상자 3개
  • 응모 횟수 30회 - 혼이 나간 지렁이 3개
  • 응모 횟수 50회 - 천사의 낚싯대 30일
  • 응모 횟수 70회 - 봉인된 염라의 신줏단지 1개
  • 응모 횟수 120회 - 차사의 영혼 주머니 1개
  • 닉네임 응모와 관련 유의사항은 상단 '이벤트 유의사항'을 확인해 주세요.
  • 닉네임 응모 횟수 추가 보상 아이템은 조건 달성 시 1계정당 1회 지급되며, 중복 지급되지 않습니다.
  • 닉네임 응모 횟수 추가 보상 아이템은 선물함으로 지급되며, 선물함 보관 만료 기간은 2주입니다.
  • 부정한 방법으로 이벤트 참여 및 아이템 수령 시 약관 및 정책에 따라 제재 혹은 회수될 수 있습니다.

이벤트 관련 변경 및 추가 안내사항

  • 4차(8/31)까지 응모 가능한 닉네임 목록이 전체 공개로 변경되었습니다.(단, 응모는 안내된 일자에 가능)
  • 닉네임 응모 당첨 시 개별 안내를 통해 변경 희망 동의 절차가 진행되며, 변경 비희망 시 다음 순위 당첨자에게 변경 기회가 주어집니다.
  • 닉네임 응모 당첨자는 9/6(화) 발표되며, 최종 변경 대상자의 변경 일자는 추후 안내될 예정입니다.
  • 응모 이벤트에 포함되지 않는 휴면 닉네임의 재사용 가능 일정은, 9월 중 검색 기능과 함께 공개될 예정입니다.
